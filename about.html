About Cataract Detector

With the aid of Convolutional Neural Network Cataract Detector was developed to classify between cataract and non-cataract infected eyes. The web application was developed by using Flask, a popular full-stack web development framework written in python. The application was first built on the localmachine. Then, it was hosted on Heroku. The user him-self or his companion can take a snapshot of the userâ€™s eyes using any camera. The captured photos will need to be uploaded to the application. Basing on the prediction of the model on the uploaded image the result is displayed in the User Interface as Cataract or Non-cataract.
